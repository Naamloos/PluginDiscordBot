Your Discord bot can browse files! send cat pictures! talk with you!

A simple Discord bot that can simply be modified by adding some Plugins.

Easily add more functions to your bot

Simply add a DDL file to your "plugins" folder and you're done!

Easy to set up

At first run, a config file will be set up. just fill in your info and run the bot again!

Too easy?

It's so easy to use, even your 12 year old minecraft youtuber brother could set it up!.
